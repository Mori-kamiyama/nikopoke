#  ゲームの流れ

0. 戦うボタンを押す
1. 六匹のポケモンのセットを選択する。
2. マッチを作る
3. マッチに参加する
4. お互いのポケモンを見て六匹セットから三匹ポケモンを選択する＋最初のポケモンを選ぶ

5. バトルが始まる
6. 決着がつく
7. リザルトが出る
8. ホームに戻る



# バトルの流れ
1. ポケモンが出る (最初、相手・自分のポケモンがいない時)
2. 技(たたかう)か交代(ポケモン)を選択
    2-1. たたかうを選択
        2-1-1. ポケモンが覚えている4つの技から一つを選ぶ
    2-2. ポケモンを選択
        2-2-1. 交代するポケモンを選択
        2-2-2. ポケモンを交代
        2-2-3. 内部的には技なしとしてバトルを実行す
        
3. ポケモンどうしのSpeedを比較し速いじゅんに技を実行する(一部Speedを無視して必ず先行になる技がある　両方とも先制技の場合は優先度が高いものが先に、そうでない場合はスピードが高いものが先にでる)
4. 倒れた・強制的に交代させる技がでた場合
    4-1. 倒れた場合
        4-1-1. 残ったポケモンからプレイヤーが選択して交代する
    4-2. 強制的に交代させる技が当たった場合
        4-2-1. 残ったポケモンからランダムに選んで交代する
5. どちらかが出せるポケモンがなくなるまで1~4を繰り返す
6. 倒れたポケモンの数が三匹になった場合
    6-1. 勝敗を判定する
