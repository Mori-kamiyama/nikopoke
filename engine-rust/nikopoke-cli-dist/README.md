# 🎮 ニコポケ バトルCLI

ポケモン風バトルシミュレーター

## インストール方法

### 1. ダウンロードして解凍
ZIPファイルを解凍してください。

### 2. ⚠️ 初回起動時の注意（重要！）

Macのセキュリティで「開発元が未確認」と表示される場合：

**方法A: ターミナルで実行（おすすめ）**
```bash
cd nikopoke-cli-dist
xattr -cr .
./battle-cli
```

**方法B: 右クリックで開く**
1. Finderで `battle-cli` を右クリック
2. 「開く」を選択
3. 警告が出たら「開く」をクリック

**方法C: システム設定から許可**
1. `battle-cli` をダブルクリック（ブロックされる）
2. システム設定 > プライバシーとセキュリティ
3. 「このまま開く」をクリック

## 使い方

### バトルモード
```bash
./battle-cli
```
ポケモンを選んで対戦できます！

### デバッグモード
```bash
./debug-cli
```
ポケモンや技の情報を確認できます。

## 操作方法

- 数字キーで選択
- `/help` でコマンド一覧
- `/moves` で技の詳細
- `/status` でステータス確認
